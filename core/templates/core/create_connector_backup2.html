⚠️ Requisito Importante - Sincronización Incremental

Para que la sincronización funcione correctamente, todas las tablas deben contener una columna updated_at de tipo TIMESTAMP.

Su servicio debe actualizar este campo automáticamente. Cada vez que se modifique un registro, asegúrese de incluir:

updated_at = NOW()
Ejemplo: UPDATE tabla SET campo = 'valor', updated_at = NOW() WHERE id = 1;
